## 简介

大多数时候，你都应该直接去官网找视频
https://developer.apple.com/videos/all-videos/

但是官网有时突然就下线了某个视频，没有任何征兆。因此，这个仓库更多的是记录一些已经不在官网保留的视频。

---

这里还有一个网站，维护了历史WWDC相关的PDF文件，所以如果你想找找之前的WWDC视频，也可以先在这里找到标题，然后再根据标题去Google

https://docs.huihoo.com/apple/wwdc/
